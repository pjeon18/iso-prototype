05 · How to use this kit

From here to hi-fi

1. Pull the feature inventory into your backlog, filtering by MVP first.
2. Recreate the screen map as your Figma page structure / flow.
3. Build each wireframe at hi-fi using the candlelit design system (plum / ember / amber, Fraunces + Inter) — the element list in each caption is your spec.
4. Prototype the layout variants for Home, Live room, and Keep-talking, and validate the recommended options.
5. Reference ISO_Brand_and_UX.html, ISO_HiFi_Mockups.html, and ISO_Prototype.html for the hi-fi target.
